# Support

MikaVN Library is a local-first Windows desktop app. When asking for help, please include enough context to reproduce the issue without sharing private data.

## Before Filing An Issue

- Check `README.md` for setup and supported workflows.
- Check `ROADMAP.md` for current boundaries and deferred features.
- Check `PRIVACY.md` before sharing logs or screenshots.
- Try the latest release or latest `main` build if practical.

## Useful Diagnostics

Include:

- Windows version.
- MikaVN Library version.
- Whether the issue happens in Tauri desktop mode, browser preview mode, or both.
- The action that failed and the visible error message.
- Redacted diagnostic log snippets if the Settings log preview shows relevant entries.

Avoid sharing:

- API keys or tokens.
- Full private Windows user profile paths.
- Private game installation paths unless they are necessary and redacted.
- Game assets or screenshots that you do not have permission to redistribute.

## Feature Requests

Feature requests are welcome. Please describe the workflow you want to improve, the current workaround, and whether the request affects local data safety.
